# 💤 Jarago – 수면 루틴 기록 앱

**Jarago**는 사용자가 자는 시간과 일어나는 시간을 손쉽게 기록하고,  
수면 시간을 자동으로 계산하여 매일의 수면 루틴을 시각화해주는 iOS 앱입니다.  
귀여운 거북이 캐릭터 "자라(Jara)"와 함께 수면 습관을 만들어보세요!

---

## 🚀 주요 기능 (MVP 기준)

### 🌙 취침 기록
- 앱 내 "잘게요" 버튼을 길게 (2초) 누르면 취침 시간이 기록됩니다.
- 설정한 수면 목표 시간(예: 8시간)을 기준으로 아침 알림이 자동 설정됩니다.

### ☀️ 기상 기록
- 아침 알림을 탭하여 앱을 열면 자동으로 기상 시간이 기록됩니다.
- 별도 "일어났어요" 버튼은 없습니다.

### 🧮 수면 시간 계산
- 취침/기상 시간 간의 차이를 자동 계산하여 수면 시간으로 저장합니다.

### 📊 기록 시각화
- 당일 수면 시간은 일간 그래프로 표시됩니다.
- 수면 기록은 날짜별 리스트 형태로 확인할 수 있습니다.
